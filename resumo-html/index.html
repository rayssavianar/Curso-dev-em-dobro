Meu resumo de HTML

Hierarquia de texto

o texto é feito por hierarquia, sendo elas Títulos,subtítulos e parágrafos

Títulos

A tag <h1>, ela serve pra ser o tema principal da página, aquela que chama mais atenção

Subtítulos

É um "título" sua tag geralmete é <h2, h3, h4...>mais completo para fornercer mais insight ou suporte ao título

Parágrafos

É onde esta o texto, tem um maior número de informações, tag usada <p>

Formatação de texto

Texto em negrito, tag <strong>

Texto em itálico, tag <i>

Texto em sublinhado, tag <u>

Texto em taxado, tag <s>

Comentário

É um texto que o leitor não consegue ver, apenas é visível no código, pressione ctrl + ;

Links externos

Quando se usa um link de outro site no seu site, tag usada <a> recomendado usar _blank para que seu usuário possa retornar para sua página

Links internos

Ao criar um novo index.html, voce pode fazer uma conexão cologando o index depois da tag a

Imagens

É possível por uma imagem no seu site, colocando <img>, ela pode ser tanto interna como externa

Listas

Você pode usar tanto uma lista ordenada <ol> como uma lista não ordenada <ul>
